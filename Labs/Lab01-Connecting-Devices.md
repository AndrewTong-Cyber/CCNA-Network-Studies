# Lab 01 - Connecting Devices

**Topics:** Connecting devices, cable types, 
straight-through vs crossover, single mode vs multimode fiber

**File:** My first CCNA Lab.pkt

**What I built:** This was my first time using Packet Tracer. 
I built a simple network with routers, switches, and servers 
and connected them using the correct cable types. I learned 
that crossover cables are used when connecting the same types 
of devices and straight-through cables are used when connecting 
different types of devices. I also practiced choosing between 
single mode and multimode fiber depending on the distance. 
I did not configure the inside of any devices yet as this 
lab was focused on getting comfortable with Packet Tracer 
for the first time.
